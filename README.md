# ⚡ Budget Management System — Electrical Panel Maintenance

Web application developed to replace a manual budgeting process in an electrical services company, providing greater control, speed and reliability.

---

## Notice

Due to company confidentiality requirements, the source code is not publicly available. This repository contains project documentation, architecture details, screenshots and technical information about the solution.

---

## Features

* Multiple users operating simultaneously without conflicts
* Real-time cost calculation on the frontend
* Dashboard with charts and budget status tracking
* Excel report export
* Complete budget management system

---

## Technologies

### Backend

* ASP.NET Core
* Entity Framework Core

### Frontend

* React

### Database

* PostgreSQL

### Others

* ClosedXML (Excel generation)
* IIS (Windows Server deployment)

---

## Problem Solved

Before:

* Manual process
* High risk of errors
* Low productivity

After:

* Process automation and error reduction
* Easier information sharing
* Faster budget generation workflow
